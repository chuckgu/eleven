# Eleven FC: AI 네이티브 축구 경험

## 1. 비전 선언문
- 모든 경기가 페르소나 기반 인공지능에 의해 구동되는 살아있는 시뮬레이션인 최초의 완전 AI 네이티브 축구 타이틀 제공
- 미리 만들어진 애니메이션과 스크립트 기반 전술을 넘어 11개의 자율 플레이어 모델이 만들어내는 창발적 결과 렌더링
- 라이브 데이터, 커뮤니티 피드백, 자가 학습 에이전트를 통해 지속적으로 진화하는 시스템 제공

## 2. 핵심 플레이어 판타지
- 독특한 AI 페르소나로 구성된 스쿼드를 관리하고, 발전시키며, 감정적으로 연결
- 각 경기가 새로운 이야기를 들려주는 인터랙티브 내러티브로서의 축구 경험
- 조이스틱 조작을 미세하게 관리하는 대신 고수준 전략, 훈련 집중도, 심리적 준비에 영향 행사

## 3. 전통적인 축구 게임과의 차별화 요소
| 전통적인 EA 스타일 축구 게임 | Eleven FC |
| --- | --- |
| 사전 제작된 애니메이션과 물리 엔진 | 페르소나 기반 의사결정 그래프가 창발적 시뮬레이션을 제공하는 실시간 시스템 |
| 버튼 입력을 통한 플레이어 조작 | 전술 슬라이더와 상황별 명령을 통한 감독 지시 |
| 정적인 선수 속성 | 멘탈, 전술, 기술, 상황 레이어를 가진 동적 페르소나 모델 |
| 고정된 경기장 장면의 3D 렌더링 캐릭터 | AI 생성 방송 패키지: 경기 비주얼, 해설, 하이라이트 |

## 4. 페르소나 시스템 개요
- **페르소나 축**
  - **멘탈리티**: 침착성, 회복력, 리더십, 공격성
  - **전술 지능**: 공간 인식, 패턴 인식, 플레이북 숙달도
  - **기술 능력**: 볼 컨트롤, 패스 범위, 마무리 다양성, 수비 기술
  - **상황별 상태**: 컨디션, 피로도, 사기, 환경 적응력
- **라이프사이클**
  1. **생성**: 실제 경기 데이터셋을 활용한 대규모 학습을 통해 기초 페르소나 벡터 생성
  2. **진화**: 경기 결과가 강화학습 및 선호도 모델에 피드백
  3. **개인화**: 훈련 선택과 내러티브 이벤트를 통해 플레이어 결정이 페르소나 파라미터에 반영

## 5. 경기 시뮬레이션 루프
1. 경기 전 브리핑에서 전술적 의도와 감정적 톤 형성
2. 각 페르소나가 상황별 상태 샘플링 (피로도, 자신감, 날씨 영향)
3. 페르소나 그래프가 미시적 의도 결정 (압박, 위치 유지, 위험한 패스 시도)
4. 멀티-에이전트 시뮬레이션 엔진이 약 10Hz 의사결정 속도로 상호작용 해결
5. 프레임 수준 이벤트가 자연어 설명으로 변환
6. 텍스트 파이프라인이 생성형 이미지 모델을 구동하여 방송 품질의 스틸/짧은 클립 생성
7. 라이브 해설 LLM이 렌더링된 시퀀스와 동기화하여 경기 해설

## 6. 감독 상호작용 레이어
- **전술 캔버스**: 영역, 행동, 트리거 조건을 정의하는 역할 타일을 드래그 앤 드롭
- **감정 다이얼**: 페르소나 샘플링에 영향을 주는 팀 멘탈리티 설정 (침착함, 공격적, 표현적)
- **적응형 피드백**: LLM 어시스턴트가 인사이트 제공 ("왼쪽 측면에 오버로드 형성 중")
- **내러티브 선택**: 라커룸 대화, 미디어 대응, 선수 멘토링이 페르소나 사기에 영향

## 7. 진행 & 라이브 운영
- **시즌 아크**: 진화하는 라이벌 관계와 스토리라인이 있는 절차적 생성 리그
- **페르소나 성장**: 훈련 선택, 경험, 감정적 이정표에 의해 구동되는 스킬 트리
- **라이브 데이터 연결**: 실제 경기 데이터를 선택적으로 수집하여 주간 챌린지 시드 생성
- **커뮤니티 랩**: 플레이어가 커스텀 페르소나 조합을 실험하고 전술을 공유하는 샌드박스 모드

## 8. 기술 스택 (기초)
- **시뮬레이션 코어**: 훈련 루프를 위한 미분 가능한 컴포넌트를 가진 Rust 또는 C++ 에이전트 플랫폼
- **페르소나 모델**: 전술/멘탈 추론을 위한 Mixture-of-Experts 트랜스포머; 확산 기반 스킬 실행 모듈
- **콘텐츠 생성**: 경기장 환경에 최적화된 멀티모달 파이프라인 (텍스트 → 스토리보드 → 이미지/비디오)
- **인프라**: 시뮬레이션, 렌더링, 라이브 운영 대시보드를 오케스트레이션하는 클라우드 네이티브 마이크로서비스

## 9. 로드맵 (Phase 0-2)
- **Phase 0 — 프로토타입 (0-3개월)**
  - 페르소나 입력을 받는 간소화된 2D 경기 시뮬레이터 구축
  - 주요 경기 순간을 위한 경량 확산 이미지 생성기 통합
  - 웹 대시보드 프로토타입을 통한 감독 상호작용 루프 검증
- **Phase 1 — 버티컬 슬라이스 (3-9개월)**
  - 전체 스쿼드 관리 기능으로 페르소나 확장
  - 기본 해설 LLM 및 하이라이트 릴 생성기 구현
  - 참여도 및 내러티브 영향에 대한 텔레메트리를 포함한 비공개 알파 진행
- **Phase 2 — 라이브 베타 (9-18개월)**
  - 지속적인 리그 및 시즌 콘텐츠를 위한 인프라 확장
  - 커뮤니티 랩 및 라이브 데이터 통합 출시
  - 저지연 스트리밍을 위한 렌더링 파이프라인 최적화

## 10. 다음 단계
- 페르소나 데이터 스키마 및 훈련 데이터셋 구체화
- 내러티브 참여도와 전술적 깊이를 포착하는 텔레메트리 메트릭 정의
- 감독 대시보드 UX 플로우 및 감정 피드백 루프 프로토타입 제작
